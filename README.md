## üìå How To Install?

-  Clone This Repo

```
$ git clone https://github.com/elyasprba/el-CoffeeShop-Frontend.git
```

-  Go To Folder Repo

```
$ cd el-coffeeshop
```

-  Setup .env

```
REACT_APP_HOST = "https://el-coffee-shop.herokuapp.com"
```

-  Install Module

```
$ npm install
```

-  To Start The Project

```
$ npm run start
```
