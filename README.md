## üìå How To Install?

-  Clone This Repo

```
git clone https://github.com/elyasprba/el-CoffeeShop-Frontend.git
```

-  Go To Folder Repo

```
cd el-coffeeshop
```

-  Install Module

```
npm install
```

-  Type `npm run start` To Start The Project
-  Set file .env `http://localhost:8080` from backend
